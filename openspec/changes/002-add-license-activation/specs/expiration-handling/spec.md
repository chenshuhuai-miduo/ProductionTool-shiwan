# 规范：过期处理和提醒

## ADDED Requirements

### REQ-EXPIRATION-001: 应用启动时过期检查

#### Scenario: 启动时检查授权状态
**Given** 应用启动  
**When** 系统初始化  
**Then** 系统应：
- 读取本地许可证文件
- 验证许可证有效性
- 确定授权状态
- 根据授权状态执行相应处理：
  - 未激活：显示未激活提示对话框（可关闭）
  - 试用已过期：显示试用过期对话框（不可关闭）
  - 正式已过期：显示正式过期对话框（不可关闭）
  - 试用期内/已激活：正常启动，检查是否需要显示到期提醒

**实现要求**：
- 在`MiduoApplicationLauncher`或`MiduoFrontendApplication`中实现
- 过期对话框必须在启动时显示
- 过期对话框不可关闭（必须续期或退出）

### REQ-EXPIRATION-002: 试用到期提醒弹窗

#### Scenario: 试用剩余≤3天时显示提醒
**Given** 授权状态为"试用期内"  
**When** 剩余天数≤3天  
**Then** 系统应：
- 界面样式参考D:\MiDuoCode\productiontool\productiontools\202510产线采集关联软件\202510产线采集关联软件\DEV\ActiveCode\trial-expire.html

### REQ-EXPIRATION-005: 未激活提示弹窗

#### Scenario: 首次启动未激活提示
**Given** 应用首次启动且无许可证文件  
**When** 系统初始化  
**Then** 系统应显示未激活提示弹窗：
- 软件图标/Logo
- 标题："欢迎使用产线采集软件！"
- 说明文字：" 您已获得15天免费试用期"
  - 试用期内可使用全部功能：
  -✅ 启用/停用生产任务
  -✅ 设备连接和数据采集
  -✅ 数据上传
  -✅ 码查询和码替换
- 试用到期后功能将不可用，如需继续使用请联系管理员获取激活码
- [立即激活]按钮
- [开始使用]按钮
- 弹窗可关闭（点击×或"开始使用"）

**实现要求**：
- 界面实现参考D:\MiDuoCode\productiontool\productiontools\202510产线采集关联软件\202510产线采集关联软件\DEV\ActiveCode\trial-welcome.html
- 弹窗可关闭
- 点击"立即激活"进入离线激活向导
- 点击"开始使用"关闭弹窗（进入15天的试用期）
