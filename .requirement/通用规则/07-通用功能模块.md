# 通用功能模块

> 本文档描述赋码采集关联系统产品线各产线软件**共用**的功能模块，开发时直接复用，不重复设计；产线差异在各产线文档中单独补充。

---

## 一、激活码（许可证管理）

**已在 T1.1.0 完成，后续产线集成接入。**

| 项目 | 说明 |
| ---- | ---- |
| **功能** | 软件需激活才能使用核心功能（采集、上传、系统设置等）；支持试用期和正式版 |
| **状态枚举** | 未激活、试用中（剩余X天）、正式激活（剩余X天）、试用过期、正式版过期 |
| **功能拦截** | 关键入口（采集、上传、系统配置等）均由 `LicenseGuard` 统一校验，未激活/过期弹窗提示并拒绝访问 |
| **到期提醒** | 剩余 ≤30天时自动弹窗提醒续期 |
| **状态栏展示** | 右下角实时展示授权状态（颜色区分：>30天绿色、30-7天橙色、<7天红色、过期灰色），圆角框突出 |
| **入口** | 菜单「帮助」→「关于系统」或弹窗内「立即激活」按钮 |

## 二、设备连接管理

| 项目 | 说明 |
| ---- | ---- |
| **功能** | 统一管理各类采集设备（相机、扫码枪、报警灯、剔除装置等）的连接状态 |
| **状态展示** | 状态栏左侧实时展示设备连接状态；断线时报警信息区显示具体原因 |
| **协议支持** | RS232、RS485、TCP/IP、Modbus |
| **开始采集前检测** | 点击「开始采集」时自动检测所有配置设备是否在线，未连接则阻止采集并提示 |
| **断线处理** | 采集中断线记录日志并在报警区提示，不自动停止采集 |

## 三、操作日志

| 项目 | 说明 |
| ---- | ---- |
| **功能** | 记录系统关键操作，供管理员事后查询追溯 |
| **记录字段** | 操作人、模块、操作类型、操作内容、操作结果、时间、备注 |
| **操作类型** | 新增、修改、删除、查询、启用、停用、采集、关联、替换、导入、导出、登录、退出等 |
| **写入时机** | 业务操作完成后异步批量写入，不阻塞主流程（`OperateLogBatchManager`） |
| **查询入口** | 菜单「数据」→「操作日志」；支持按日期范围筛选，分页展示（默认20条/页） |
| **查看方式** | 列表行悬浮 Tooltip 展示详情，无需跳转详情页 |

---

## 四、退出软件

| 项目 | 说明 |
| ---- | ---- |
| **触发** | 菜单「文件」→「退出软件」；或点击窗口关闭按钮 |
| **采集中判断** | 若正在采集，提示「请先停止采集」，停止后再执行退出判断 |
| **未满垛判断**（有未满垛概念的产线） | 弹窗「存在未满垛(X箱)，是否暂存？」，三个选项：**暂存**（存库标记未成垛，允许退出）、**强制满垛**（生成垛码后退出）、**取消**（返回主界面） |
| **正常退出** | 无未满垛时弹窗确认后退出 |

## 五、密码保护

| 项目 | 说明 |
| ---- | ---- |
| **适用操作** | 系统设置、取消关联、码替换等重要/危险操作 |
| **机制** | 访问时弹窗输入密码，每次访问均需验证，**不保存登录态** |
| **密码错误** | 提示重新输入，不限次数 |
| **密码管理** | 暂时固定为 `123456`，后续迭代再考虑可配置化 |

## 六、帮助提示（? 图标）

| 项目 | 说明 |
| ---- | ---- |
| **形态** | 圆圈问号图标 `(?)`，浅灰色，14~16px，紧跟标题或按钮组末尾 |
| **触发** | 点击显示浮层/弹窗，列举该区域所有按钮的功能说明 |
| **消失** | 点击浮层外部或按 ESC 关闭 |
| **适用位置** | 任务控制区、多按钮操作组等功能密集区域 |
| **样式** | 白底、1px浅灰边框、轻微阴影 |

## 八、操作二次确认机制

| 项目 | 说明 |
| ---- | ---- |
| **适用场景** | 不可逆操作：删除、取消关联、强制满垛、退出等 |
| **弹窗内容** | 明确说明操作影响范围（如「将解除X条关联，操作不可恢复」） |
| **密码叠加** | 高危操作（取消关联、码替换、系统设置）在二次确认基础上额外要求输入密码 |
| **按钮规范** | 确认按钮用高亮/危险色（红色）；取消按钮用灰色；文字18px以上加粗 |
