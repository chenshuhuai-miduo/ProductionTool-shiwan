# 码包数据使用场景整理

> 整理自 V1.2 石湾产线采集关联需求文档，供码包相关调整时参考。

## 一、码包类型与对应关系


| 码包类型  | 对应码层级   | 用途           |
| ----- | ------- | ------------ |
| 盖外码小标 | 瓶码（盖外码） | 校验瓶码是否在码包范围内 |
| 盒外码中标 | 盒码      | 校验盒码是否在码包范围内 |
| 箱外码大标 | 箱码      | 校验箱码是否在码包范围内 |

## 二、使用场景汇总

### 1. 数据采集 - 实时码有效性校验


| 所属      | 页面/环节                      | 使用码包        | 用途                              |
| ------- | -------------------------- | ----------- | ------------------------------- |
| **1号机** | P01-01 主界面-瓶盒关联（环节2 瓶码采集）  | 盖外码小标       | 校验每个瓶码是否在小标码包范围内；不在则错码、不计数、标记剔除 |
| **1号机** | P01-01 主界面-瓶盒关联（环节3 盒码关联）  | 盒外码中标       | 校验盒码是否在中标码包范围内；不在则错码、不计数、标记剔除   |
| **2号机** | P02-01 主界面-盒箱垛关联（环节4 箱码关联） | 箱外码大标       | 校验箱码是否在大标码包范围内；不在则错码、不计数、剔除     |
| **2号机** | P02-02 手工采集（瓶盒关联备用）        | 盖外码小标、盒外码中标 | 手工扫码瓶盒关联时，同样需校验瓶码、盒码在对应码包范围内    |


**说明**：1号机、2号机共用同一数据库（部署在2号机），码包数据由 P02-03 码包管理导入后，两端均可读取。

### 3. 码包管理（2号机）


| 所属      | 页面          | 使用码包 | 用途                                                                           |
| ------- | ----------- | ---- | ---------------------------------------------------------------------------- |
| **2号机** | P02-03 码包管理 | 全部   | **删除校验**：仅当该码包内无任何已关联码时允许删除；有已关联码则禁止（码替换时新码需在码包内校验）                          |
| **2号机** | P02-03 码包管理 | 全部   | **查看**：弹窗展示该码包下所有码，支持搜索与分页                                                   |
| **2号机** | P02-03 码包管理 | 全部   | **重复判断**：导入时检测码包内数据是否与已导入重复或高度重叠；**冷热分离时须同时查热表+冷表**（或统一视图 all_codes），任一存在即重复 |

### 5. 剔除记录与报警信息


| 所属          | 场景         | 使用码包 | 用途                                       |
| ----------- | ---------- | ---- | ---------------------------------------- |
| **2号机**     | 剔除记录表、报警信息 | 间接   | 剔除原因枚举含「小标不通过」「中标不通过」「大标不通过」；问题码为不在码包内的码 |
| **1号机/2号机** | 数据接收区、操作日志 | 间接   | 错码时显示「xxx 不在码包范围内」等具体原因                  |


---

### 6. 取消关联（2号机）— 间接影响


| 所属      | 页面          | 与码包关系    | 说明                                                                                                |
| ------- | ----------- | -------- | ------------------------------------------------------------------------------------------------- |
| **2号机** | P02-06 取消关联 | **间接影响** | 取消关联**不校验**码包，但取消关联后，原已关联的码变为未使用状态，会**影响码包删除条件**：该码包内「已关联码」数量减少，若全部解除则可能变为可删除；**冷热分离时需将码从冷表回迁热表** |


**说明**：取消关联扫的是已有关联的码，用于识别并取消关联，无需查码包。但删除码包时校验「该码包内无任何已关联码」，取消关联会改变这一条件。**冷热分离**：取消关联后需将码**回迁**热表。

## 三、冷热数据分离下的数据源（推荐设计）

采用**热表 + 冷表**分离时，各场景查询的数据源如下：


| 场景            | 数据源         | 说明                            |
| ------------- | ----------- | ----------------------------- |
| 采集校验、数据替换新码校验 | **热表**      | 只查未使用码                        |
| 码包导入重复判断      | **热表 + 冷表** | 须同时查，任一存在即重复；或建统一视图 all_codes |
| 取消关联后         | **回迁**      | 取消关联的码从冷表回迁热表，方可参与替换          |
| 码包查看、删除边界     | 热表 + 冷表     | 按需分 Tab 或合并查询                 |

## 七、说明

- **码位数配置**（系统设置）校验的是采码的**位数**是否符合配置，不依赖码包内的具体码值。
- **米多兜底**：无包材厂回传时，米多作为兜底方校验并告知；软件侧仍依赖本地码包做采集时校验。
