# P02-04 - 数据查询

## 子功能名称

数据查询 - 码关联信息查询

## 功能概述

为生产管理人员和质量控制人员提供专业的码查询功能界面，采用桌面应用风格设计，与系统整体风格保持一致。界面提供完整的码查询和关联信息展示功能，输入码后一次性查询该码的所有关联关系（上级、同级、下级），分层展示在左侧表格，选中行后在右侧展示详情，适合生产线码追溯和质量管理场景。

**重要说明**：本功能参考已有功能（功能名称为"查询码"），采用左右分栏布局，左侧分层表格展示所有关联数据，右侧展示选中行的详情。

**查询说明**：

- **单一查询**：输入码后执行查询，查出该码所属垛的全部数据（垛-箱-盒-瓶完整关联）
- **查询范围**：输入任意层级码（瓶/盒/箱/垛），均显示该码所属垛内的所有数据。例如输入箱码1，显示垛1下的箱1、箱2、箱3…及各自的盒、瓶
- **分层展示**：左侧表格按层级分列展示，每行一条完整关联链
- **输入码标识**：表格中与输入的码对应的单元格使用红色字体（#F44336）显示
- **详情联动**：鼠标选中表格某行后，右侧展示该行的具体信息

**包装层级说明**：
四级包装结构：瓶 → 盒 → 箱 → 垛

## 界面布局设计

### 整体布局

- **布局方式**：桌面应用风格，全屏显示，左右分栏
- **适配要求**：全宽布局，充分利用屏幕空间
- **主要区域**：
  - **顶部**：码输入框、查询按钮、清空按钮、帮助按钮、查询状态
  - **左侧**：码关联信息表格（约2/3宽度），分层展示所有关联数据
  - **右侧**：具体信息详情区（约1/3宽度），展示选中行的详细信息

### 操作按钮栏设计

- **布局结构**：顶部横条，左侧搜索组、右侧状态
- **搜索组元素**：
  - **码输入框**：
    - 功能：支持手动输入和扫码枪输入码信息
    - 样式：输入框高度40px，文字18px
    - 占位符："请输入码信息或使用扫码枪扫描..."
    - 最大长度：50字符
    - 支持回车键快速查询
  - **查询按钮**：
    - 功能：执行查询，查出该码所有关联关系
    - 样式：主要操作按钮，蓝色背景，白色加粗文字
    - 文字：查询
  - **清空按钮**：
    - 功能：清空输入框和查询结果
    - 样式：标准操作按钮
    - 操作：点击清空所有内容
  - **查询帮助按钮（？）**：
    - 功能：显示查询帮助信息弹窗
    - 样式：帮助按钮，带"？"字样，圆形按钮
- **查询状态区域**：
  - 默认状态："未查询"
  - 查询中状态："正在查询..."
  - 查询成功："查询成功"（绿色文字），并显示"已查询到 X 条码信息"
  - 查询失败："未找到该码信息"（红色文字）

## 左侧：码关联信息表格

### 功能说明

- **标题**：码关联信息
- **空状态**：未查询时显示"请输入码进行查询"
- **有数据时**：显示"已查询到 X 条码信息"，下方为分层表格

### 表格设计

- **表格列**（从左到右）：
  - **序号**：行号，灰色文字(#999)，20px字体
  - **第一层（瓶码）**：瓶码值，黑色文字
  - **第二层（盒码）**：盒码值，黑色文字
  - **第三层（箱码）**：箱码值，黑色文字
  - **第四层（垛码）**：垛码值，黑色文字
- **行数据说明**：每行代表一条完整关联链，各列填充该层级对应的码值；若该行在该层级无数据则留空
- **输入码标识**：与输入的码对应的单元格使用红色字体（#F44336）显示，便于快速定位
- **表格行高**：不小于52px
- **分页或滚动**：数据量大时支持表格滚动，显示总数量"已查询到 X 条码信息"

## 右侧：具体信息详情区

### 功能说明

- **标题**：具体信息
- **空状态**：未选中行时显示"请点击左侧记录查看详细信息"
- **选中行后**：展示该行的详细信息

### 详情内容设计

- **码详细信息**：
  - 瓶码、盒码、箱码、垛码（有则显示，无则显示"-"）
  - 采集时间：YYYY-MM-DD HH:mm:ss格式
  - 是否关联：已关联/未关联
- **产品信息**：
  - 产品编号
  - 产品名称
  - 生产单号

## 查询功能设计

### 查询方式

- **手动输入**：支持键盘输入码信息
- **扫码输入**：支持扫码枪直接输入（自动触发查询）

### 查询触发

- **点击查询按钮**：执行查询
- **回车键**：执行查询
- **扫码枪输入后**：自动触发查询

### 查询逻辑

- 输入任意层级码（瓶/盒/箱/垛），查出该码所属垛的全部数据
- 查询范围：以垛为根，显示该垛内所有箱、盒、瓶的完整关联。例如输入箱码，显示该箱所属垛下的所有箱（箱1、箱2…）及各自的盒、瓶
- 结果按层级分列展示：第一层（瓶码）、第二层（盒码）、第三层（箱码）、第四层（垛码）
- 与输入的码对应的单元格使用红色字体显示

### 查询结果处理

- **成功查询**：
  - 左侧表格展示所有关联数据，输入码对应单元格红色字体
  - 状态栏显示"查询成功"及"已查询到 X 条码信息"
- **查询失败**：
  - 显示"未找到该码信息"（红色文字）
  - 左侧显示空状态"请输入码进行查询"
- **网络异常**：
  - 显示"查询服务异常"（红色文字）
  - 提供重试机制

## 交互功能设计

### 表格行选中与详情联动

- **选中行**：鼠标点击左侧表格某行，该行选中（可选高亮样式）
- **详情更新**：右侧详情区实时展示选中行的具体信息（码、采集时间、产品编号、产品名称、生产单号）
- **输入码标识**：与输入的码对应的单元格使用红色字体（#F44336）显示

### 清空操作交互

- **清空按钮**：点击清空按钮后
  - 清空输入框内容
  - 清空左侧表格数据
  - 右侧详情区恢复为空状态提示
  - 状态栏恢复到"未查询"
- **确认清空**：如有查询结果，可选择弹出确认对话框

### 帮助信息交互

- **帮助按钮**：点击"？"按钮显示帮助对话框
- **关闭方式**：点击对话框外部或关闭按钮关闭帮助

## 功能边界

### 包含的功能

1. 单一查询：输入码后查出该码所属垛的全部数据（垛-箱-盒-瓶）
2. 分层展示：左侧表格按层级分列（第一层瓶码、第二层盒码、第三层箱码、第四层垛码）
3. 输入码标识：表格中与输入的码对应的单元格红色字体显示
4. 详情联动：选中表格某行后，右侧展示码详细信息（含采集时间）、产品信息（产品编号、产品名称、生产单号）
5. 支持大数据量展示（表格滚动）
6. 扫码枪输入支持

### 适用场景

- **生产线追溯**：产品生产过程追溯
- **质量管理**：产品质量信息查询
- **仓储管理**：库存码信息查询
