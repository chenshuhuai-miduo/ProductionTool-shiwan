# 前端开发文档

## 1. 概述

本文档描述产线采集关联软件的前端实现，包括UI组件、交互逻辑、数据流、设备通信等。前端采用JavaFX框架，遵循MVC架构模式，使用FXML进行界面布局。

## 2. 技术架构

### 2.1 技术栈
- **框架**: JavaFX
- **布局**: FXML
- **架构模式**: MVC（Model-View-Controller）
- **通信方式**: HTTP API（HttpUtil）
- **线程模型**: JavaFX Application Thread + 后台线程

### 2.2 项目结构

```java
private void onEnableTask() {
    if (currentTask == null) {
        showAlert(Alert.AlertType.WARNING, "提示", "请先选择生产订单");
        return;
    }
    
    Integer newStatus = currentTask.getOrderStatus() == 1 ? 0 : 1;

        new TypeReference<ApiResult<Boolean>>() {});
    
    if (result.getCode() == 200) {
        // 更新任务状态
        currentTask.setOrderStatus(newStatus);
        updateEnableButton();
        updateButtonStates();
    } else {
        showAlert(Alert.AlertType.ERROR, "错误", result.getMessage());
    }
}
```

#### 3.2.3 箱码采集
**功能描述**:
- 接收箱码采集设备的数据
- 根据采集类型（有箱码/无箱码）处理数据
- 无箱码模式时，箱码必须为"null"，否则提示错误
- 有箱码模式时，检查重复码
- 采集成功后，更新当前箱数统计

**实现逻辑**:
```java
private void handleSecondDeviceData(String boxCode, String deviceName) {
    if (currentTask == null) {
        return;
    }
    
    // 获取当前采集类型
    Integer currentType = typeComboBox.getSelectionModel().getSelectedIndex() == 0 ? 1 : 2;
    
    // 无箱码模式：箱码必须为"null"
    if (currentType == 2) {
        if (!"null".equals(boxCode)) {
            String message = getCurrentTime() + " [" + deviceName + "] 收到非null值: " + boxCode + "，无箱码模式忽略";
            operationLogArea.appendText(message + "\n");
            alarmInfoArea.appendText(message + "\n");
            return;
        }
    }
    
    // 构建采集请求
    CodeCollectRequest request = new CodeCollectRequest();
    request.setOrderNo(currentTask.getOrderNo());
    request.setBoxCode(boxCode);
    request.setProductNo(currentTask.getProductNo());
    request.setBatchNo(productionBatchField.getText());
    request.setType(currentType);
    
    // 调用采集API
    String responseJson = HttpUtil.doPost("/api/code/collect", request);
    ApiResult<CodeCollectResult> result = HttpUtil.parseJson(responseJson, 
        new TypeReference<ApiResult<CodeCollectResult>>() {});
    
    if (result.getCode() == 200) {
        CodeCollectResult collectResult = result.getData();
        // 处理采集结果
        handleCollectResult(collectResult, boxCode, deviceName);
    }
}
```

#### 3.2.4 托盘码关联
**功能描述**:
- 接收托盘码关联设备的数据（托盘码）
- 使用之前保存的触发箱码进行关联
- 有箱码模式：调用`/api/code/associate-pallet`接口
- 无箱码模式：调用`/api/code/collect-no-box`接口
- 关联成功后，更新统计信息和实时上传数据区

**实现逻辑**:
```java
private void handleFirstDeviceData(String palletCode, String deviceName) {
    if (currentTask == null) {
        return;
    }
    
    // 获取当前采集类型
    Integer currentType = typeComboBox.getSelectionModel().getSelectedIndex() == 0 ? 1 : 2;
    
    if (currentType == 1) {
        // 有箱码模式：托盘码关联
        PalletAssociateRequest request = new PalletAssociateRequest();
        request.setOrderNo(currentTask.getOrderNo());
        request.setPalletCode(palletCode);
        request.setTriggerBoxCode(triggerBoxCode);
        request.setQty(boxesPerPallet);
        
        String responseJson = HttpUtil.doPost("/api/code/associate-pallet", request);
        ApiResult<PalletAssociateResult> result = HttpUtil.parseJson(responseJson, 
            new TypeReference<ApiResult<PalletAssociateResult>>() {});

            handleAssociateResult(result.getData(), palletCode, deviceName);

    } else {

        NoBoxCollectRequest request = new NoBoxCollectRequest();

        request.setPalletCode(palletCode);
        request.setBoxesPerPallet(boxesPerPallet);
        
        String responseJson = HttpUtil.doPost("/api/code/collect-no-box", request);
        ApiResult<NoBoxCollectResult> result = HttpUtil.parseJson(responseJson, 
            new TypeReference<ApiResult<NoBoxCollectResult>>() {});
        
        if (result.getCode() == 200) {
            handleNoBoxCollectResult(result.getData(), palletCode, deviceName);
        }
    }
}
```

#### 3.2.6 实时统计更新
**功能描述**:
- 定时查询当前箱数、已生产垛数、完成度
- 当前箱数：从内存缓存和数据库查询（无箱码模式需要查询CodeRelationUpload表）
- 已生产垛数：查询CodeRelationUpload表中BigSerialNumber有值的不同TagNo数量
- 完成度：已采集箱数 / 计划数量 * 100%

**实现逻辑**:
```java
private void updateStatisticsDisplay(boolean updateCurrentBoxCount) {
    if (updateCurrentBoxCount) {
        currentBoxCountLabel.setText(String.valueOf(currentBoxCount));
    }
    boxesPerPalletLabel.setText(String.valueOf(boxesPerPallet));
    
    // 从后端API获取已生产垛数和完成度
    if (currentTask != null) {
        String orderNo = currentTask.getOrderNo();
        if (orderNo != null && !orderNo.isEmpty()) {
            new Thread(() -> {
                try {
                    // 1. 获取已生产垛数
                    String palletCountJson = HttpUtil.doGet("/api/code/produced-pallet-count/" + orderNo);
                    ApiResult<Integer> palletCountResult = HttpUtil.parseJson(palletCountJson, 
                        new TypeReference<ApiResult<Integer>>() {});
                    
                    if (palletCountResult.getCode() == 200 && palletCountResult.getData() != null) {
                        int dbProducedPalletCount = palletCountResult.getData();
                        Platform.runLater(() -> {
                            producedPalletCount = dbProducedPalletCount;
                            producedPalletCountLabel.setText(String.valueOf(dbProducedPalletCount));
                        });
                    }
                    
                    // 2. 获取已采集箱数并计算完成度
                    String collectedCountJson = HttpUtil.doGet("/api/code/collected-count/" + orderNo);
                    ApiResult<Integer> collectedCountResult = HttpUtil.parseJson(collectedCountJson, 
                        new TypeReference<ApiResult<Integer>>() {});
                    
                    if (collectedCountResult.getCode() == 200 && collectedCountResult.getData() != null) {
                        int collectedCount = collectedCountResult.getData();
                        double completionRate = (double) collectedCount / currentTask.getProductCount() * 100;
                        
                        Platform.runLater(() -> {
                            completionRateLabel.setText(String.format("%.1f%%", completionRate));
                        });
                    }
                } catch (Exception e) {
                    System.err.println("获取统计数据失败：" + e.getMessage());
                }
            }).start();
        }
    }
}
```

```java
private void updateRealTimeUploadArea() {
    if (currentTask == null) {
        uploadDataArea.getChildren().clear();
        return;
    }
    
    // 从后端API获取实时上传数据
    String orderNo = currentTask.getOrderNo();
    if (orderNo != null && !orderNo.isEmpty()) {
        new Thread(() -> {
            try {
                String responseJson = HttpUtil.doGet("/api/upload/realtime/" + orderNo);
                ApiResult<List<RealTimeUploadVO>> result = HttpUtil.parseJson(responseJson, 
                    new TypeReference<ApiResult<List<RealTimeUploadVO>>>() {});
                
                if (result.getCode() == 200 && result.getData() != null) {
                    List<RealTimeUploadVO> uploadList = result.getData();
                    Platform.runLater(() -> {
                        uploadDataArea.getChildren().clear();
                        for (RealTimeUploadVO upload : uploadList) {
                            Label label = createUploadLabel(upload);
                            uploadDataArea.getChildren().add(0, label);  // 最新记录在上方
                        }
                    });
                }
            } catch (Exception e) {
                System.err.println("获取实时上传数据失败：" + e.getMessage());
            }
        }).start();
    }
}
```

```java
public class DeviceConnectionManager {
    private static DeviceConnectionManager instance;
```
