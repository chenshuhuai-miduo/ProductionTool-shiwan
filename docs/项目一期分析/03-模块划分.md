# 模块划分

> 产线采集关联软件 - Maven模块结构说明

---

## 3.1 Maven模块结构

项目采用**Maven多模块**结构，共8个子模块：

| 模块名 | 职责 | 说明 |
|--------|------|------|
| **miduo-common** | 公共模块 | 通用DTO、工具类 |
| **miduo-entity** | 实体模块 | DTO、PO、枚举定义 |
| **miduo-domain** | 领域层 | 领域模型、领域服务、仓储接口 |
| **miduo-infrastructure** | 基础设施层 | 仓储实现、MyBatis Mapper |
| **miduo-application** | 应用层 | 应用服务、用例编排 |
| **miduo-controller** | 控制器层 | RESTful API接口 |
| **miduo-frontend** | 前端模块 | JavaFX应用 |
| **miduo-bootstrap** | 启动模块 | Spring Boot启动类 |

---

## 3.2 模块依赖关系

```
miduo-bootstrap (启动模块)
    ├── miduo-controller
    │   ├── miduo-application
    │   │   ├── miduo-domain
    │   │   │   └── miduo-entity
    │   │   └── miduo-infrastructure
    │   │       ├── miduo-entity
    │   │       └── miduo-common
    │   └── miduo-common
    └── miduo-frontend (独立启动)
        └── miduo-common
```

---

## 3.3 各模块详细说明

### 3.3.1 miduo-common（公共模块）

**位置**：`miduo-common/src/main/java/com/miduo/cloud/common/`

**主要类**：
- `ApiResult<T>` - 统一API响应封装
- `PageInput` - 分页查询输入
- `PageOutput<T>` - 分页查询输出
- `IniFileUtil` - INI文件工具类

**职责**：提供跨模块的通用工具类和DTO定义

---

### 3.3.2 miduo-entity（实体模块）

**位置**：`miduo-entity/src/main/java/com/miduo/cloud/entity/`

**主要包结构**：
```
entity/
├── dto/              # DTO对象
│   ├── code/         # 码相关DTO（23个类）
│   ├── task/         # 任务相关DTO
│   ├── device/       # 设备相关DTO
│   ├── operatelog/   # 操作日志DTO
│   └── upload/        # 上传相关DTO
├── enums/            # 枚举类型
│   ├── LogLevel.java
│   ├── ModuleNameEnum.java
│   └── OperateTypeEnum.java
├── model/            # 模型类
│   └── LogEntry.java
└── po/               # 持久化对象
    └── OperateLog.java
```

**职责**：定义所有数据传输对象（DTO）、持久化对象（PO）和枚举类型

---

### 3.3.3 miduo-domain（领域层）

**位置**：`miduo-domain/src/main/java/com/miduo/cloud/domain/`

**主要包结构**：
```
domain/
├── task/             # 任务领域
│   ├── model/        # Task领域模型
│   ├── repository/   # TaskRepository接口
│   └── service/      # 任务领域服务
├── code/             # 码领域
│   ├── model/        # CodeRelation领域模型
│   ├── repository/   # CodeRepository接口
│   └── service/      # 码领域服务
└── device/           # 设备领域
    ├── model/        # IoDevice领域模型
    └── repository/   # IoDeviceRepository接口
```

**核心领域模型**：
- `Task` - 生产任务领域模型
- `CodeRelation` - 码关联领域模型
- `IoDevice` - IO设备领域模型

**职责**：定义业务领域模型和业务规则，不依赖基础设施

---

### 3.3.4 miduo-infrastructure（基础设施层）

**位置**：`miduo-infrastructure/src/main/java/com/miduo/cloud/infrastructure/`

**主要包结构**：
```
infrastructure/
├── persistence/
│   └── mybatis/
│       ├── mapper/   # MyBatis Mapper接口
│       │   ├── CodeRelationMapper.java
│       │   ├── ProductionOrderMapper.java
│       │   ├── ProductionOrderDetailMapper.java
│       │   └── IoDeviceMapper.java
│       └── po/       # 持久化对象
│           ├── CodeRelationPO.java
│           ├── ProductionOrderPO.java
│           ├── ProductionOrderDetailPO.java
│           └── IoDevicePO.java
├── mapper/           # MyBatis Mapper（操作日志）
│   └── OperateLogMapper.java
└── repository/       # 仓储实现
    ├── CodeRepositoryImpl.java
    ├── TaskRepositoryImpl.java
    ├── IoDeviceRepositoryImpl.java
    └── OperateLogRepositoryImpl.java
```

**职责**：实现领域层的仓储接口，提供数据持久化能力

---

### 3.3.5 miduo-application（应用层）

**位置**：`miduo-application/src/main/java/com/miduo/cloud/application/`

**主要服务类**：
- `TaskApplicationService` - 任务应用服务
- `CodeApplicationService` - 码应用服务
- `DataUploadApplicationService` - 数据上传应用服务
- `DeviceApplicationService` - 设备应用服务
- `OperateLogApplicationService` - 操作日志应用服务
- `WmsUploadApplicationService` - WMS上传应用服务

**特殊组件**：
- `CodeBloomFilterManager` - 布隆过滤器管理器（用于码去重）
- `BloomFilterInitListener` - 布隆过滤器初始化监听器

**职责**：编排领域服务，处理用例流程，管理事务边界

---

### 3.3.6 miduo-controller（控制器层）

**位置**：`miduo-controller/src/main/java/com/miduo/cloud/controller/`

**主要控制器**：
- `TaskController` - 任务管理API（9个接口）
- `CodeController` - 码管理API（20+个接口）
- `DataUploadController` - 数据上传API
- `DeviceController` - 设备管理API
- `OperateLogController` - 操作日志API
- `WmsUploadController` - WMS上传API

**配置类**：
- `WebMvcConfig` - Web MVC配置
- `GlobalExceptionHandler` - 全局异常处理

**职责**：定义RESTful API接口，处理HTTP请求响应

---

### 3.3.7 miduo-frontend（前端模块）

**位置**：`miduo-frontend/src/main/java/com/miduo/cloud/frontend/`

**主要包结构**：
```
frontend/
├── controller/       # JavaFX控制器（16个）
│   ├── MainController.java          # 主界面控制器
│   ├── TaskManagementController.java # 任务管理
│   ├── ProductManagementController.java # 产品管理
│   ├── DataUploadController.java     # 数据上传
│   ├── CodeQueryController.java      # 码查询
│   ├── CodeReplaceController.java    # 码替换
│   └── SystemConfigController.java   # 系统配置
├── service/          # 前端服务层（5个）
│   ├── TaskService.java
│   ├── ProductService.java
│   ├── CodeService.java
│   ├── DataUploadService.java
│   └── SystemConfigService.java
├── util/            # 工具类（8个）
│   ├── HttpUtil.java              # HTTP请求工具
│   ├── OperateLogBatchManager.java # 操作日志批量管理
│   ├── FileLogManager.java        # 文件日志管理
│   └── ...
└── config/          # 配置类
    └── FrontendConfig.java
```

**资源文件**：
- `resources/fxml/` - FXML界面文件（15个）
- `resources/css/` - CSS样式文件（7个）
- `resources/icons/` - 图标资源

**职责**：提供JavaFX桌面应用界面，通过HTTP调用后端API

---

### 3.3.8 miduo-bootstrap（启动模块）

**位置**：`miduo-bootstrap/src/main/java/com/miduo/cloud/`

**主要类**：
- `MiduoBackendApplication` - 后端Spring Boot启动类
- `MiduoApplicationLauncher` - 应用启动器
- `GlobalExceptionHandler` - 全局异常处理
- `MyBatisConfig` - MyBatis配置

**配置文件**：
- `application.properties` - 主配置文件
- `application-device-only.properties` - 设备专用配置

**职责**：提供Spring Boot应用启动入口和全局配置
