# 注意事项

> 赋码采集关联系统 - 注意事项说明

---

## 12.1 数据库兼容性

- 项目使用JTDS驱动连接SQL Server
- SQL语法需要兼容MySQL和SQL Server
- 注意日期时间类型的处理差异

---

## 12.2 前端后端分离

- 前端JavaFX应用独立启动，不依赖Spring Boot
- 通过HTTP调用后端API
- 需要确保后端服务已启动

---

## 12.3 设备通信

- 支持串口和网口两种连接方式
- 需要根据实际设备配置连接参数
- 设备连接状态需要实时监控

---

## 12.4 性能优化

- 使用布隆过滤器减少数据库查询
- 操作日志批量提交减少HTTP请求
- 连接池配置优化数据库连接
