# 项目概述

> 产线采集关联软件 - 项目概述

---

## 1.1 项目背景

这是一个**产线采集关联软件**，用于制造业生产线的数据采集和追溯。主要解决以下问题：

- **技术自主可控**：摆脱对第三方供应商的依赖
- **数据追溯**：实现生产过程全程追溯
- **自动化采集**：自动采集各层级码关联关系
- **数据集成**：与标识平台对接，确保数据一致性

---

## 1.2 项目定位

- **应用类型**：桌面应用程序（JavaFX） + 后端服务（Spring Boot）
- **部署环境**：Windows工控机
- **运行模式**：前后端分离，前端JavaFX独立运行，后端提供RESTful API
- **数据库**：MySQL 8.0+（SQL Server兼容）

---

## 1.3 核心价值

- ✅ 支持多层级码关联（箱码、托盘码、垛码等）
- ✅ 实时数据采集和上传
- ✅ 完整的操作日志记录
- ✅ 设备IO控制集成
- ✅ 码查询、替换、删除等数据维护功能
