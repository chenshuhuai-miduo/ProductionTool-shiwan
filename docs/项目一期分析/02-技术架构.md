# 技术架构

> 产线采集关联软件 - 技术架构说明

---

## 2.1 整体架构

项目采用**前后端分离**架构：

```
┌─────────────────────────────────────────────────────────┐
│                    Windows工控机环境                    │
│                                                           │
│  ┌──────────────────┐         ┌──────────────────┐      │
│  │  JavaFX前端应用  │  HTTP   │  Spring Boot后端  │      │
│  │  (miduo-frontend)│  ←──→   │  (miduo-bootstrap)│      │
│  │  独立启动        │         │  端口: 8080      │      │
│  └──────────────────┘         └──────────────────┘      │
│         │                              │                 │
│         │                              │                 │
│         └──────────────┬───────────────┘                 │
│                        │                                 │
│                 ┌──────▼──────┐                          │
│                 │   MySQL     │                          │
│                 │  数据库     │                          │
│                 └─────────────┘                          │
└─────────────────────────────────────────────────────────┘
```

---

## 2.2 分层架构

项目采用**DDD（领域驱动设计）**分层架构：

```
┌─────────────────────────────────────────────────────────┐
│  Controller层 (miduo-controller)                        │
│  - RESTful API接口定义                                   │
└─────────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────┐
│  Application层 (miduo-application)                       │
│  - 应用服务编排                                          │
│  - 事务管理                                              │
└─────────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────┐
│  Domain层 (miduo-domain)                                 │
│  - 领域模型（Task, CodeRelation等）                      │
│  - 领域服务                                              │
│  - 仓储接口                                              │
└─────────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────┐
│  Infrastructure层 (miduo-infrastructure)                 │
│  - 仓储实现（MyBatis）                                   │
│  - 数据持久化                                            │
└─────────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────┐
│  Entity层 (miduo-entity)                                 │
│  - DTO对象定义                                           │
│  - PO对象定义                                            │
│  - 枚举类型                                              │
└─────────────────────────────────────────────────────────┘
```

## 2.3 前端架构

前端采用**JavaFX + FXML**架构：

```
┌─────────────────────────────────────────────────────────┐
│  JavaFX前端 (miduo-frontend)                             │
│                                                           │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐  │
│  │  Controller  │  │    Service   │  │     Util     │  │
│  │  (FXML绑定)  │  │  (业务逻辑)  │  │  (工具类)    │  │
│  └──────────────┘  └──────────────┘  └──────────────┘  │
│         │                  │                  │          │
│         └──────────────────┼──────────────────┘          │
│                            │                             │
│                    ┌───────▼───────┐                    │
│                    │   HttpUtil    │                    │
│                    │  (HTTP请求)   │                    │
│                    └───────┬───────┘                    │
│                            │                             │
│                    ┌───────▼───────┐                    │
│                    │ 后端API服务    │                    │
│                    │ (localhost:8080)│                   │
│                    └────────────────┘                    │
└─────────────────────────────────────────────────────────┘
```
