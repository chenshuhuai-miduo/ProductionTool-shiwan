## 5.1 核心数据表

### 5.1.1 ProductionOrder（生产订单表）

**表结构**：
```sql
CREATE TABLE ProductionOrder (
    Id INT AUTO_INCREMENT PRIMARY KEY,
    SrcId INT NOT NULL COMMENT '原订单表id',
    Type TINYINT NOT NULL COMMENT '类型 (1: 有箱码, 2: 无箱码)',
    OrderNo VARCHAR(255) NULL COMMENT '订单编号',
    OrderSumCoun INT NOT NULL COMMENT '生产总数量',
    ProductSumCount INT NOT NULL COMMENT '订单总数量',
    DmakeDate DATETIME NOT NULL COMMENT '制单日期',
    OrderStatus INT NOT NULL COMMENT '订单状态 (0: 待入库, 1: 入库中, 2: 入库完成, 5: 提前结单)',
    ProductSourceType INT NOT NULL COMMENT '产品来源类型 (0: 默认, 1: PDA, 2: 产线)',
    CreateTime DATETIME NOT NULL,
    LastModifyTime TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    ProductionLine INT NOT NULL COMMENT '所属生产线'
);
```

**说明**：存储生产订单主信息

---

### 5.1.2 ProductionOrderDetail（生产订单明细表）

**表结构**：
```sql
CREATE TABLE ProductionOrderDetail (
    Id INT AUTO_INCREMENT PRIMARY KEY,
    OrderCount INT NOT NULL COMMENT '已生产数量',
    ProductCount INT NOT NULL COMMENT '计划订单数量',
    ProductID INT NOT NULL COMMENT '产品ID',
    ProductName VARCHAR(255) NULL COMMENT '产品名称',
    ProductNO VARCHAR(255) NULL COMMENT '产品编号',
    ProductForMatID INT NOT NULL COMMENT '规格ID',
    ProductForMatName VARCHAR(255) NULL COMMENT '规格名称',
    OrderNo VARCHAR(255) NULL COMMENT '订单编号',
    OrderId INT NOT NULL COMMENT '订单ID',
    SyBatchNo VARCHAR(255) NULL COMMENT '批次信息',
    Type INT NOT NULL COMMENT '类型 1 有箱码 2 无箱码',
    Ratio INT NOT NULL COMMENT '拖箱比例',
    IsDel INT NOT NULL COMMENT '是否删除 1删除',
    AddTime DATETIME NOT NULL,
    ProductTime DATETIME NOT NULL COMMENT '生产时间',
    TwillendTime DATETIME NOT NULL COMMENT '预计完工时间',
    CreateTime DATETIME NOT NULL,
    LastModifyTime TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

**说明**：存储生产订单明细信息，一个订单可以包含多个产品

### 5.1.3 CodeRelationUpload（码关系上传表）

**表结构**：
```sql
CREATE TABLE CodeRelationUpload (
    ID INT AUTO_INCREMENT PRIMARY KEY,
    BiggerSerialNumber VARCHAR(50) NOT NULL COMMENT '跺标',
    BigSerialNumber VARCHAR(50) NOT NULL COMMENT '大标（托盘码）',
    MediumSerialNumber VARCHAR(50) NOT NULL COMMENT '中标',
    SmallSerialNumber VARCHAR(50) NOT NULL COMMENT '小标（箱码）',
    DxCode VARCHAR(255) NOT NULL COMMENT '动销码',
    SalesCode VARCHAR(255) NOT NULL COMMENT '导购码',
    VirtualSerialNumber VARCHAR(50) NOT NULL COMMENT '虚拟跺标',
    IsVirtual INT NOT NULL DEFAULT 0 COMMENT '是否虚拟标签；0否；1是',
    ProductNO VARCHAR(50) NULL COMMENT '产品编号',
    OrderNo VARCHAR(50) NULL COMMENT '订单编号',
    Status INT NOT NULL DEFAULT 0 COMMENT '状态 0未完成 1已完成 2已解除',
    TagNo VARCHAR(255) NOT NULL COMMENT '标签编号',
    Qty INT NOT NULL DEFAULT 0 COMMENT '标签数量（托盘数量）',
    Type INT NOT NULL COMMENT '类型 1有箱码 2无箱码',
    WarehouseNo VARCHAR(50) NULL COMMENT '仓库编号',
    BatchNo VARCHAR(50) NULL COMMENT '批次号',
    AddTime DATETIME NOT NULL,
    ErrCount INT NOT NULL DEFAULT 0 COMMENT '失败次数',
    Msg VARCHAR(2000) NULL COMMENT '备注',
    UploadTime DATETIME NULL COMMENT '上传时间',
    IsUpload INT NOT NULL DEFAULT 1 COMMENT '是否同步（默认0）0同步 1不同步',
    IsDel INT NOT NULL DEFAULT 0 COMMENT '是否删除',
    TeamName VARCHAR(255) NULL COMMENT '班组名称'
);
```

**说明**：存储码关联关系，支持多层级码关联（箱码、托盘码、垛码等）

### 5.1.4 OperateLog（操作日志表）

**表结构**：
```sql
CREATE TABLE OperateLog (
    ID BIGINT AUTO_INCREMENT PRIMARY KEY,
    OperatorName VARCHAR(50) NOT NULL COMMENT '操作人姓名',
    OperatorIp VARCHAR(50) NULL COMMENT '操作人IP地址',
    ModuleName VARCHAR(100) NOT NULL COMMENT '操作模块',
    Type INT NOT NULL COMMENT '1 操作日志 2 系统日志',
    OperateType VARCHAR(50) NOT NULL COMMENT '操作类型',
    TargetId VARCHAR(50) NULL COMMENT '操作对象ID',
    TargetName VARCHAR(100) NULL COMMENT '操作对象名称',
    OperateContent TEXT NULL COMMENT '操作内容',
    BeforeData TEXT NULL COMMENT '操作前数据(JSON)',
    AfterData TEXT NULL COMMENT '操作后数据(JSON)',
    OperateResult VARCHAR(20) NOT NULL COMMENT '操作结果',
    FailReason TEXT NULL COMMENT '失败原因',
    OperateTime DATETIME(3) NOT NULL DEFAULT CURRENT_TIMESTAMP(3),
    DeviceInfo VARCHAR(200) NULL COMMENT '操作设备',
    Remark VARCHAR(500) NULL COMMENT '备注信息'
);
```

**索引**：
- `idx_operatelog_modulename` - 模块名索引
- `idx_operatelog_operatetime` - 操作时间索引
- `idx_operatelog_operatetype` - 操作类型索引

## 5.2 数据关系

```
ProductionOrder (1) ──< (N) ProductionOrderDetail
                              │
                              │ (通过OrderNo关联)
                              │
                              ▼
                    CodeRelationUpload (N)
                              │
                              │ (通过OrderNo和ProductNo关联)
                              │
                              ▼
                         OperateLog (N)
```
