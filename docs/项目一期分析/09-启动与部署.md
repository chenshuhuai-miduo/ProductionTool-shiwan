# 启动与部署

> 产线采集关联软件 - 启动与部署指南

---

## 9.1 后端启动

**启动类**：`MiduoBackendApplication`

**启动方式**：
```bash
# 方式1：直接运行main方法
java -jar miduo-bootstrap-1.0.0-SNAPSHOT.jar

# 方式2：Maven启动
cd miduo-bootstrap
mvn spring-boot:run
```

**默认端口**：8080

**访问地址**：`http://localhost:8080/api`

---

## 9.2 前端启动

**启动类**：`MiduoFrontendApplication`

**启动方式**：
```bash
# 方式1：直接运行main方法
java -jar miduo-frontend-1.0.0-SNAPSHOT.jar

# 方式2：JavaFX Maven插件启动
cd miduo-frontend
mvn javafx:run
```

**窗口大小**：1400x900

**后端地址配置**：在`HttpUtil.java`中配置后端API地址（默认：`http://localhost:8080`）

---

## 9.3 数据库配置

**配置文件**：`miduo-bootstrap/src/main/resources/application.properties`

**关键配置**：
```properties
# 数据源配置
spring.datasource.url=jdbc:jtds:sqlserver://192.168.5.22:1433/cshTest_db
spring.datasource.username=test2016
spring.datasource.password=test2016
spring.datasource.driver-class-name=net.sourceforge.jtds.jdbc.Driver

# 连接池配置
spring.datasource.hikari.minimum-idle=10
spring.datasource.hikari.maximum-pool-size=50
```

**注意**：项目使用JTDS驱动连接SQL Server，兼容MySQL语法

---

## 9.4 设备配置

**配置文件**：`io_devices.ini`

**配置格式**：
```ini
[设备名称]
type=设备类型
connection=连接方式
ip=IP地址
port=端口号
...
```
