# 项目结构

> 米多赋码采集关联系统 - 项目目录结构说明

---

## 7.1 目录结构

```
productiontools/
├── pom.xml                          # 父POM文件
├── .gitignore                       # Git忽略配置
├── io_devices.ini                   # IO设备配置文件
├── logs/                            # 日志目录
│   └── 2025-11/                    # 按月份分组的日志
│
├── miduo-common/                    # 公共模块
│   └── src/main/java/.../common/
│       ├── dto/                    # 通用DTO
│       └── util/                   # 工具类
│
├── miduo-entity/                    # 实体模块
│   └── src/main/java/.../entity/
│       ├── dto/                    # DTO定义
│       ├── enums/                  # 枚举类型
│       ├── model/                  # 模型类
│       └── po/                     # 持久化对象
│
├── miduo-domain/                    # 领域层
│   └── src/main/java/.../domain/
│       ├── task/                   # 任务领域
│       ├── code/                   # 码领域
│       └── device/                 # 设备领域
│
├── miduo-infrastructure/           # 基础设施层
│   └── src/main/java/.../infrastructure/
│       ├── persistence/mybatis/   # MyBatis实现
│       └── repository/            # 仓储实现
│
├── miduo-application/               # 应用层
│   └── src/main/java/.../application/
│       ├── task/                   # 任务应用服务
│       ├── code/                   # 码应用服务
│       ├── device/                 # 设备应用服务
│       └── ...
│
├── miduo-controller/                # 控制器层
│   └── src/main/java/.../controller/
│       ├── TaskController.java
│       ├── CodeController.java
│       └── ...
│
├── miduo-frontend/                  # 前端模块
│   └── src/main/
│       ├── java/.../frontend/
│       │   ├── controller/         # JavaFX控制器
│       │   ├── service/            # 前端服务
│       │   └── util/               # 工具类
│       └── resources/
│           ├── fxml/               # FXML界面文件
│           ├── css/                # 样式文件
│           └── icons/              # 图标资源
│
├── miduo-bootstrap/                 # 启动模块
│   └── src/main/
│       ├── java/.../
│       │   └── MiduoBackendApplication.java
│       └── resources/
│           └── application.properties
│

```
