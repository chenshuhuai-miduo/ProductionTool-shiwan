## 🚀 5分钟快速了解

### 项目是什么？

这是一个**产线数据采集和追溯系统**，用于制造业生产线：
- 📦 自动采集产品码（箱码、托盘码、垛码等）
- 🔗 建立多层级码关联关系
- 📤 上传数据到第三方系统
- 📊 提供完整的追溯查询功能

### 技术架构

```
JavaFX前端 (桌面应用)
    ↕ HTTP
Spring Boot后端 (RESTful API)
    ↕ MyBatis-Plus
MySQL数据库
```

### 核心模块

| 模块 | 说明 | 启动类 |
|------|------|--------|
| **miduo-frontend** | JavaFX桌面应用 | `MiduoFrontendApplication` |
| **miduo-bootstrap** | Spring Boot后端 | `MiduoBackendApplication` |
| **miduo-application** | 应用服务层 | - |
| **miduo-domain** | 领域模型层 | - |
| **miduo-controller** | RESTful API | - |
| **miduo-infrastructure** | 数据持久化 | - |
| **miduo-entity** | DTO/PO定义 | - |
| **miduo-common** | 公共工具 | - |

## 🔑 核心功能

### 1. 任务管理
- 创建/修改/删除生产任务
- 任务状态管理（启用/停用/完成）
- 任务查询和筛选

### 2. 码采集
- **第一台设备**：读码剔除控制
- **第二台设备**：箱码采集
- **第三/四台设备**：托盘码关联
- 支持有箱码/无箱码两种模式

### 3. 数据上传
- 一键批量上传
- 单个订单上传
- 上传状态跟踪

### 4. 码查询与维护
- 码关联查询（多层级）
- 码替换
- 码删除

### 5. 设备管理
- IO设备配置
- 设备连接测试
- 设备状态监控

## 🛠️ 如何启动

### 1. 启动后端

```bash
# 方式1：运行启动类
运行 MiduoBackendApplication.main()

# 方式2：Maven命令
cd miduo-bootstrap
mvn spring-boot:run
```

**默认端口**：8080  
**访问地址**：http://localhost:8080/api

### 2. 启动前端

```bash
# 方式1：运行启动类
运行 MiduoFrontendApplication.main()

# 方式2：Maven命令
cd miduo-frontend
mvn javafx:run
```

**窗口大小**：1400x900
