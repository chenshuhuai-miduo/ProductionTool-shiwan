## 项目完整结构报告

### 1. Maven 模块结构

根 pom.xml (`miduo-ccas`) 包含以下 **10 个子模块**，分为核心层和变体层：

| 模块 | 说明 |
|------|------|
| `miduo-common` | 公共工具类、DTO基类（ApiResult、PageInput/Output） |
| `miduo-entity` | 实体/DTO层（请求体、VO、枚举等） |
| `miduo-infrastructure` | 基础设施层（数据库访问、MyBatis Mapper） |
| `miduo-domain` | 领域层（Domain Model、Repository接口、DomainService） |
| `miduo-application` | 应用层（ApplicationService、BloomFilter等） |
| `miduo-controller` | Web控制器层（Spring MVC REST接口） |
| `miduo-frontend` | **JavaFX前端模块**（FXML、CSS、Controller、Service） |
| `miduo-bootstrap` | 启动模块（Spring Boot启动类、全局异常、MyBatis配置） |
| `miduo-variant-zhimeizhai` | 变体层 - 至美宅（完整业务实现：persistence+application+controller） |
| `miduo-variant-shiwan` | 变体层 - 石湾（目前只有空占位 `ShiwanVariantConfig`） |

### 3. CSS 文件结构

全部位于 `miduo-frontend/src/main/resources/css/`，采用 **分文件组织** 方式，`base-styles.css` 通过 `@import` 统一引入：

```
base-styles.css      ← 入口文件（@import其余所有CSS）
theme-colors.css     ← 主题色彩系统（CSS变量定义）
typography.css       ← 字体系统（字体族、字号、字重变量）
button-styles.css    ← 按钮样式（primary/secondary/danger/success/toolbar）
input-styles.css     ← 输入控件样式（text-field/combo-box/text-area）
table-styles.css     ← 表格样式（TableView完整定制）
status-styles.css    ← 状态指示器（许可证状态/任务状态颜色）
liceseInfo-style.css ← 许可证信息界面专属样式
expired-style.css    ← 过期弹窗专属样式
deviceVerification-style.css ← 设备验证专属样式
renewal-success.css  ← 续期成功专属样式
```

**CSS 设计规范要点：**

- **字体**：全局使用 `Microsoft YaHei`（微软雅黑）
- **主色调**：`#4a90e2`（蓝色），CSS变量 `-primary-color`
- **尺寸系统**：字号 small(11px) / normal(14px) / large(16px) / xlarge(18px) / xxlarge(20px)
- **按钮最小高度**：36px（toolbar按钮70px）
- 所有 FXML 均通过 `stylesheets="@../css/base-styles.css"` 引入样式

### 5. 主界面 (MainWindow.fxml) 结构

主界面为 `BorderPane` 布局，控制器为 `MainController`：

- **top**：菜单栏（文件/配置/数据/帮助） + 工具栏（任务管理/数据上传/查询码/码替换/清屏）
- **center**：三栏 HBox：
  - **左侧面板**（340px）：当前生产信息 + 实时上传数据区
  - **中间面板**（自适应）：数据接收区 + 操作日志 + 报警信息
  - **右侧面板**（336px）：单位实时统计（当前箱数/每垛箱数/已生产垛数/完成度）+ 任务控制按钮组
- **bottom**：状态栏（设备状态、工作状态、许可证状态圆点指示、时间、版本）

默认窗口大小：`1400 × 900`
